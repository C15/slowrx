slowrx

A decoder for Slow-Scanning Television (SSTV).

Download .zip Download .tar.gz View on GitHub

slowrx is an SSTV decoder for Linux that aims for intuitive simplicity and lack of features not needed by a shortwave listener.

Written in C99 by Oona Räisänen OH2-250 (windyoona at gmail).

Screenshot

Features

  • Mode detection from VIS code
  • Support for a multitude of modes (see modespec.c for a list)
  • Detect frequency-shifted signals – no need to fine-tune the radio
  • Automatic slant correction (compensation for sound card clock error)
  • Adaptive noise reduction
  • Decodes digital FSK ID
  • Saves received pictures as PNG

Requirements

  • Linux
  • Alsa
  • Gtk+ 3.4
  • FFTW 3
  • pnglite

And, obviously:

  • shortwave radio with SSB
  • computer with sound card
  • means of getting sound from one to the other

Future plans

  • Easy manual slant adjustments?
  • Support for Redundant Digital File Transfer (RDFT i.e. "Digital SSTV")?
  • Automatic timing-based mode detection (such as in QSSTV) in case of missed VIS?
